# v1.6.4

`v1.6.4` 是一次后台编辑器能力增强与调试入口调整版本，重点补齐文字颜色工具、优化 emoji 面板交互，并将 `/swagger` 重新接入公开入口便于调试。

如果只用一句话概括：

> `v1.6.4` 让后台 Markdown 编辑器可以直接插入彩色文字，修复暗色模式下 emoji / 下拉菜单的可点与可读性问题，并同步更新调试与回归测试。

## 本版摘要

这一版主要集中在四件事：

- 后台 Markdown 编辑器新增文字颜色工具，ByteMD 与 Milkdown 两套编辑器都可使用
- ByteMD emoji 面板重新定位与插入流程，暗色模式下弹层不再被编辑区遮挡
- Markdown 数学分隔符、彩色文字片段和编辑器工具栏增加回归测试
- split 与 all-in-one Caddy 入口重新代理 `/swagger`，方便从统一入口查看接口文档

## 相比 v1.6.3 的变化

如果你已经在使用 `v1.6.3`，这一版最直接的变化是：

- 后台写文章时，工具栏新增「文字颜色」按钮，可插入 `<span style="color:#xxxxxx">...</span>`
- ByteMD 暗色模式下，emoji 面板和工具栏下拉菜单的层级、点击区域、hover 可读性更稳定
- emoji 插入会恢复最近编辑区选区，避免选中 emoji 后插入位置漂移
- `/swagger` 不再由 Caddy 返回 404，而是代理到后端服务
- 新增一份通过 VanBlog HTTP API 发布文章的技能文档，便于后续自动化发布文章

## 主要更新

### 1. 后台编辑器新增文字颜色工具

ByteMD 工具栏新增文字颜色按钮，支持预设颜色与自定义 hex 颜色。没有选中文本时，会插入占位文本并自动选中，方便用户继续输入。

Milkdown 编辑器同步接入同一套彩色文字片段生成逻辑，保持两套编辑器输出一致。

### 2. emoji 面板交互修复

ByteMD emoji 插件重新维护当前按钮、面板和最近选区：

- 面板根据工具栏按钮位置动态定位
- 点击面板外部会关闭弹层
- 插入 emoji 前恢复最近编辑选区
- 对原生 `emoji-click` 事件增加兜底监听，避免不同 emoji picker 事件格式导致插入失败

### 3. 暗色模式弹层与菜单可读性增强

后台编辑器样式补齐了 emoji 面板、文字颜色面板、ByteMD 下拉菜单在暗色模式下的背景、边框、层级和 hover 状态。

这一版重点防止两类回归：

- 弹层显示出来但被 CodeMirror 绘制层盖住，导致鼠标点击落到编辑区
- 暗色模式下菜单 hover 行变成浅底浅字，导致内容不可读

### 4. `/swagger` 入口重新开放

split Caddy 与 all-in-one Caddy 都将 `/swagger*` 重新代理到后端服务。对应 compose e2e 和当前栈 API 测试已改为期望 `200`。

这项调整只影响接口文档入口，不改变数据库、Redis、Waline 或后台静态资源的部署拓扑。

### 5. 发布文章 API 技能文档

新增 `docs/skills/vanblog-publish-article.md`，记录通过 VanBlog 官方 HTTP API 创建、发布、更新、删除文章的请求格式与 Token 使用约束。

文档强调长期 Token 只能通过环境变量或一次性输入使用，不应写入仓库、日志或脚本。

## 配置与兼容性

这一版不引入数据库迁移，不改变环境变量，不改变 compose 服务数量。

需要注意：

- `/swagger` 现在会通过公开 Caddy 入口访问到后端 Swagger UI
- 如果你的生产环境不希望公开 Swagger，请在外层网关、防火墙或自定义 Caddy 配置中单独限制该路径
- 彩色文字输出使用内联 `<span style="color:#hex">...</span>`，前台 Markdown 渲染链路会继续按现有 HTML 支持策略处理

## docker compose 配置单

如果你使用锁版镜像部署：

```env
VANBLOG_DOCKER_REPO=kevinchina/deeplearning
VANBLOG_RELEASE_SUFFIX=v1.6.4-<image-id>
```

## 升级建议

升级到 `v1.6.4` 后，建议重点验证：

1. 后台 ByteMD 编辑器可以用文字颜色按钮插入彩色文字
2. 后台 Milkdown 编辑器可以通过工具栏插入彩色文字
3. 暗色模式下 ByteMD emoji 面板可以正常点击并插入 emoji
4. 暗色模式下 ByteMD 标题 / 代码语言下拉菜单 hover 状态可读
5. `/swagger` 通过公开入口返回 Swagger UI
6. 前台文章页能正常展示带颜色 span 的 Markdown 内容

## 回滚方法

如果线上想回滚到上一版，只需要把：

```env
VANBLOG_RELEASE_SUFFIX=v1.6.4-<image-id>
```

改回上一版的 release suffix，然后重新执行：

```bash
docker compose -f docker-compose.image.yml up -d
```
