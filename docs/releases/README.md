# 发布说明

这里收录当前仓库的重要版本说明与迁移指引。

- 默认维护分支：`master`
- tag 保持独立管理，不受分支切换影响
- 当前公开文档基线：`v1.5.5`
- AI 工作台自 `v1.4.0` 起成为正式能力，但依然保持为可选部署

- [`v1.5.5`](./v1.5.5.md)：后台文章编辑页滚动同步修复、ByteMD 预览单滚动容器收口版本
